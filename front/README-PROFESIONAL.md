## 🎯 Cómo Personalizar

### 📝 **Actualizar tu información:**
1. Edita el objeto `translations` en `PortafolioProfesional.jsx`
2. Cambia los datos de experiencia en el array `experiences`
3. Actualiza los enlaces sociales en la sección `socialHero`

### 🖼️ **Cambiar tu foto:**
1. Reemplaza `src/imagenes/MyFoto.png` con tu nueva foto
2. O actualiza la ruta en `.profileImage` del CSS

### 🎨 **Modificar colores:**
```css
/* Colores principales en PortafolioProfesional.module.css */
--primary-bg: #0a0a0b;        /* Fondo principal */
--accent-blue: #667eea;       /* Azul de acento */
--accent-purple: #764ba2;     /* Púrpura de acento */
--accent-red: #ff6b6b;        /* Rojo de acento */
```
